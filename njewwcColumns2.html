Physical width calculation for left column= content width 225 + padding left 20px + padding right 20px + border left 10px + border right 10px + margin left 20px + margin right 20px = 325px
physical width calculation for right column= content width 167px + padding left 20px + padding right 20px + margin left 20px + margin right 20px + border left 10px + border right 10px = 267px
The difference between physical width and content width is that with content width you are only accounting for the stuff that is inside of the "box", this is controlled with the width keyword in css. Physical width includes things like margin padding and border.
